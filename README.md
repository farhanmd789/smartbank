Engineered a basic secure banking platform using reactJS, Spring Boot, and
MySQL. Enabled user and admin dashboards for transactions, account management, and authentication with real
time data persistence.
### Features of this project
This project enables users to manage their finances efficiently with features such as secure money deposits and withdrawals, loan application functionality, and easy access to mini statements. The system also includes an interactive dashboard with visualizations for clear and insightful tracking of financial activity.
